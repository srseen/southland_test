# 3.

## โจทย์: ถ้า n มีค่าเท่ากับ 256 หลังจากจบการทำงานแล้ว r มีค่าเป็นเท่าใด

> 📄 ซอร์สโค้ด: [index.js](index.js)

```
i := n;
r := 0;
while (i >= 1) do
begin
    for j := 1 to n do
        r := r + 1;
    i := i / 2;
end;
```

## Trace การทำงาน

- **Outer loop**: `i` หารครึ่งทุกรอบ → 256, 128, 64, 32, 16, 8, 4, 2, 1 = **9 รอบ**
- **Inner loop**: ทุกรอบวน **256 ครั้ง** (ใช้ `n` ไม่ใช่ `i`) เพิ่ม `r` ทีละ 1

| รอบที่ | i (ก่อน)     | i (หลัง) | inner loop | r     |
| ------ | ------------ | -------- | ---------- | ----- |
| 1      | 256          | 128      | 256 ครั้ง  | 256   |
| 2      | 128          | 64       | 256 ครั้ง  | 512   |
| 3      | 64           | 32       | 256 ครั้ง  | 768   |
| 4      | 32           | 16       | 256 ครั้ง  | 1,024 |
| 5      | 16           | 8        | 256 ครั้ง  | 1,280 |
| 6      | 8            | 4        | 256 ครั้ง  | 1,536 |
| 7      | 4            | 2        | 256 ครั้ง  | 1,792 |
| 8      | 2            | 1        | 256 ครั้ง  | 2,048 |
| 9      | 1            | 0        | 256 ครั้ง  | 2,304 |
| -      | 0 < 1 → หยุด |          |            |       |

## คำตอบ: `r = 2,304`

> สูตร: r = จำนวนรอบ outer loop × n = 9 × 256 = 2,304

## ผลลัพธ์จากการ run โปรแกรม (JavaScript)

```

รอบที่ 3: i = 64 → 32, inner loop ทำ 256 ครั้ง, r = 768
รอบที่ 4: i = 32 → 16, inner loop ทำ 256 ครั้ง, r = 1024
รอบที่ 5: i = 16 → 8, inner loop ทำ 256 ครั้ง, r = 1280
รอบที่ 6: i = 8 → 4, inner loop ทำ 256 ครั้ง, r = 1536
รอบที่ 7: i = 4 → 2, inner loop ทำ 256 ครั้ง, r = 1792
รอบที่ 8: i = 2 → 1, inner loop ทำ 256 ครั้ง, r = 2048
รอบที่ 9: i = 1 → 0, inner loop ทำ 256 ครั้ง, r = 2304

=== จบการทำงาน ===
จำนวนรอบ outer loop = 9
คำตอบ: r = 2304
```
