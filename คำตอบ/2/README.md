## โจทย์: จากส่วนของโปรแกรมที่กำหนดให้ หลังจากจบการทำงานแล้ว ค่าของ i มีค่าเป็นเท่าใด

> ซอร์สโค้ด: [index.js](index.js)

```
x := 1;
i := 1;
while (x <= 1000)
begin
    x := 2^x;
    i := i + 1;
end;
```

## Trace การทำงาน

| รอบที่ | x (ก่อน)             | x := 2^x     | i   |
| ------ | -------------------- | ------------ | --- |
| 1      | 1                    | 2¹ = 2       | 2   |
| 2      | 2                    | 2² = 4       | 3   |
| 3      | 4                    | 2⁴ = 16      | 4   |
| 4      | 16                   | 2¹⁶ = 65,536 | 5   |
| -      | 65,536 > 1000 → หยุด |              |     |

## คำตอบ: `i = 5`

## ผลลัพธ์จากการ run โปรแกรม (JavaScript)

```
=== เริ่มต้น ===
x = 1, i = 1

รอบที่ 1: x = 2^1 = 2, i = 2
รอบที่ 2: x = 2^2 = 4, i = 3
รอบที่ 3: x = 2^4 = 16, i = 4
รอบที่ 4: x = 2^16 = 65536, i = 5

=== จบการทำงาน ===
คำตอบ: i = 5
```
